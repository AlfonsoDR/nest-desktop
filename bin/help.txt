A NEST desktop application
usage: nest-desktop <command>

The available commands are:
  checkport      Check if port 5000 is running
  init           Create sqlite database
  killport       Kill the PID of the service for the port 5000
  help           List available commands
  list           List module dependencies
  path           Show the path of the node module
  server-only    Start only flask server
  start          Start nest-desktop application
  test           Test the required packages for the app
  version        Show the current version
