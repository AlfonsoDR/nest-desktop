NEST Desktop

An educational GUI for neuroscience

NEST Desktop is a web-based GUI application for NEST Simulator, an advanced simulation tool for computational neuroscience.

The application enables the rapid construction, parametrization, and instrumentation of neuronal network models. The primary objective is to provide an accessible classroom tool that allows users to rapidly explore neuroscience concepts without the need to learn a simulator control language at the same time.
